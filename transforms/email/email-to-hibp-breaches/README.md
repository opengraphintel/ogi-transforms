# Email to HIBP Breaches

Looks up an email address in Have I Been Pwned and creates one aggregated evidence document summarizing all returned breaches.

## Input / Output

- **Input**: `EmailAddress`
- **Output**: `Document`

## API Keys Required

- **HIBP API key** (`HIBP_API_KEY`) — get one from the Have I Been Pwned dashboard

Configure via Settings > API Keys in the OGI UI.
