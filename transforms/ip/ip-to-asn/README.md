# IP to ASN

Looks up the Autonomous System Number (ASN) and operating organization for an IPv4 address using Team Cymru's DNS-based service.

## Input / Output

- **Input**: `IPAddress`
- **Output**: `ASNumber` (with prefix, country, registry), `Organization` (network operator)

## API Keys

None required. Uses the free Team Cymru DNS service.

## Limitations

- IPv4 only (IPv6 not yet supported)
